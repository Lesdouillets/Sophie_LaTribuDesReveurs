Resume des regles globales

Exclusions mutuelles
  • Confort : RGO+Allergie / Allergie / RGO (1 seul)
  • Espacement : Trop long / Trop court
  • Volume bib : Petit bib / Gros bib / Boit bib longtemps
  • Inversion : Jour/Nuit / Lait/Solide
  • Temps d'eveil : Trop long / Trop court
  • Siestes courtes TE OK exclut tout Temps d'eveil
  • Changement : lit / chambre / lit+chambre (1 seul)
Chaines auto-parent
  • Enfant de Courbe → Courbe de poids → Courbes pas OK
  • Trop long / Trop court → Temps d'eveil
  • Enfant Reveils matinaux → Reveils matinaux
  • Enfant Strategie → Strategie de sommeil
Dependances (requiert un frere)
  • Consequence Inconfort → necessite un frere coche
  • Consequence Mange la nuit → necessite un frere coche
Visibilite & synchronisation
  • RGO / APLV : synchronises depuis l'onglet Sante
  • Conditions de visibilite pilotees par l'onglet Sante
  • Les items auto ne sont pas cochables manuellement
  • Les items menu ouvrent un menu deroulant avec consequences
